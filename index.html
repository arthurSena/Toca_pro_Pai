Toca pro Pai

Evolução sentimental das músicas internacionais do brasil.

#Descubra

Fonte dos dados

  • Dados relativos ás 100 músicas mais tocadas nas radios brasileiras de 1958 a 2014.
  • Dados das letras das músicas vinda do site cifras.com.
  • Só utilizamos dados de músicas da lingua inglesa.

Análise de Sentimento

Análise de Sentimento refere-se ao processo de utilizar ferramentas computacionais para identificar e extrair informacões subjetivas de conteúdos textuais.

O que é polaridade ?

O quão positivo ou negativo é uma música.

Distribuição das polaridades.


Soma das polaridades por ano


Polaridade média das músicas por anos
